# Felpudo's Race

Esse é um jogo 2D desenvolvido na Unity Engine como projeto prático para a disciplina de programação de jogos no curso de jogos digitais da Universidade do Estado da Bahia e tem por objetivo trabalhar a introdução a animação de sprite e particulas visuais na Unity Engine.

# Gameplay

Jogue como Felpudo, um passarinho destemido que precisa salvar sua amada Forura que foi sequestrada pelos lacaios de Uruca, o tirano. Esquive de inimigos se movimentando com W, A e D e colete frutas que lhe concede poderes especiais para alcançar Fofura.

# Condição de vitoria

Quando sua barra de distancia se encher ao maximo quer dizer que seu destino lhe aguarda e Fofura estará logo em frente, alcance-a para finalizar o jogo e ser bem sucedido no resgate de sua amada.
